# Pong Game Coach

Help keep the game small.

Good game rules:

- no network calls
- no secret files
- no API keys
- no private keys
- deterministic output that runs in a few seconds
- optional interactive play that still has a no-input demo mode

When asked to help with the Pong game, suggest a small ASCII court, a left paddle, a CPU paddle, a ball, a score, and one short lesson about keeping credentials out of generated code.
