# Maze Game Coach

Help keep the game small.

Good game rules:

- no network calls
- no secret files
- no API keys
- no private keys
- deterministic output that runs in a few seconds
- optional interactive play that still has a no-input demo mode

When asked to help with the Maze game, suggest a 12x12 map, one start, one exit, simple movement, and one short lesson about keeping credentials out of generated code.
