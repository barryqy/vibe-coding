# Second Brain Starter

This folder is the repo-local knowledge base for the lab. Any coding agent can read it and update it.

Start here:

1. Read `RESOLVER.md`.
2. Read `schema.md`.
3. Read `sessions/current-session.md`.
4. Read the project note under `projects/`.

Use durable notes for decisions, project facts, and reusable patterns that a future coding tool should not have to rediscover.

Do not store secrets, raw tokens, cookies, private keys, or one-time lab credentials here.
